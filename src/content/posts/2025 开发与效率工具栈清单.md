# 2025 开发与效率工具栈

> 目标：启动快、切换少、体验稳定、可替换性强。下面按类别列出主力工具 + 场景速记。

## 🧠 AI 编程

- **Google Antigravity**：日常主力，内置gemini 3 pro + Claude Opus4.5模型，猛猛用起来
- **Zed**：轻量与实时协作（Zed Collaboration）。打开大仓库 & Rust/TS 项目感觉更流畅；临时阅读代码 / 快速改动时用。

## 💻 终端

- **kitty**：高性能 GPU 渲染，标签 + 分屏，支持图片显示与远程字体。自定义快捷键映射常用脚本，启动速度极快。
- **Termius**：SSH 连接管理神器，管理多台 VPS 时非常方便，支持 SFTP 与端口转发。

## 📦 包管理

- **Homebrew**：统一安装 CLI / GUI。策略：`brew bundle dump` 备份；自建 `Brewfile` 保障可复现。

## 🗄️ 数据库 & 缓存

- **DBeaver**：跨数据库客户端（Postgres / MySQL / ClickHouse / MongoDB），ER 图可视化 + 数据导出方便，免费版够用。
- **Tiny RDM**：Redis 可视化神器，TTL 观察 + JSON 结构展开，调试缓存结构清晰直观。

## 🔌 接口调试

- **Apifox**：接口文档 + Mock + 自动化测试 + 多语言代码生成，团队协作一体化平台。
- **Reqable**：HTTP/HTTPS/WebSocket/HTTP2 抓包调试，跨平台支持，定位线上问题效率高。

## 🌐 翻译（浏览器 & 系统级）

- **沉浸式翻译**：浏览器必装，段落级双语对照，阅读英文文档/论文效率翻倍。支持多翻译引擎切换。
- **Bob 社区版** + [Qwen 翻译插件](https://github.com/simongino/bob-plugin-qwen-translator)：系统级跨应用划词翻译，支持快捷键呼出。覆盖 IDE / PDF / Terminal 输出场景。

## 🐳 容器

- **OrbStack**：比 Docker Desktop 轻，启动/内存占用友好；内置轻量 VM，日常本地开发（Postgres / Redis）足够。

## 🔋 电池状态

- **battery**：命令行 / 菜单栏查看健康度、循环次数，方便判断是否需要保养。

## 🎬 视频播放

- **Infuse**：Apple 生态最佳播放器，支持杜比视界/全景声，iCloud 跨端同步观看记录。
- **IINA**：开源免费，格式兼容性强，自定义快捷键，macOS 原生设计风格。

## 📹 录屏 & 截图

- **QuickRecorder**：原生 macOS 录屏工具，支持区域/窗口/全屏录制，系统音频+麦克风音轨分离，适合录制演示或复现 Bug。
- **PixPin**：截图 + 贴图悬浮 + 长截图 + OCR 文字识别；对比设计稿或接口返回值很方便，支持像素取色与延时截图。

## 📦 解压缩

- **Keka**：macOS 解压神器，拖拽即用，支持多格式压缩/解压、加密压缩、分卷压缩。

## 🌐 FTP / 文件传输

## 🌐 FTP / 传输

- **Blip**：局域网跨设备（macOS/iOS/Android/Windows）快速传输文件，无需联网。
- **Cyberduck**：临时 SFTP / S3 / WebDAV 访问；不常驻，占用低。

## 🎯 聚焦启动 & 全局动作

- **Raycast**：Spotlight 增强替代品。自建 Script Commands（新建 Post 模板、打开常用项目、剪贴板历史、窗口管理），插件生态丰富。

## ✍️ 文本 / Markdown

- **Typora**：所见即所得 + 粘贴图片自动管理。写文档/博文草稿；最终再进仓库。

## 🖱️ 外设辅助

- **Scroll Reverser**：触控板自然滚动 + 鼠标传统滚动方向分离，外接鼠标用户必备。
- **Logi Options+**：罗技设备专用，自定义按键映射（复制路径、格式化代码、切换窗口）。

## 🧊 菜单栏管理

- **iBar**：一键折叠聚合菜单栏图标，减少视觉拥挤。
- **Barbee**：按情景（工作 / 演示 / 专注）配置不同菜单栏图标集合。

## 🔋 系统状态监控

- **Stats**：开源免费，菜单栏显示 CPU/内存/网络/磁盘/电池状态，轻量无广告。
- **Battery**：命令行 / 菜单栏查看电池健康度、循环次数，方便判断是否需要保养。

## 🧹 应用卸载 / 清理

- **Pearcleaner**：开源无跟踪，彻底清理应用残留（LaunchAgents / Caches / Preferences），告别遗留垃圾。

## 💬 AI Chat & 知识整理

- **Cherry Studio**：多模型聚合客户端（OpenAI / Claude / Gemini / 本地模型），会话分组 + Prompt 模板管理，日常 AI 对话主力。

## 🔐 密码 & 机密管理

- **1Password**：跨平台密码管理 + 自动填充 + SSH Agent（集中管理私钥，减少裸钥使用），开发者体验友好。

## 📧 邮件

- **Spark**：聚焦收件箱（智能分类），快速处理主力。
- **Filo**：简单清爽，用于干净视图（仅关注少量订阅/通知）。

## 使用策略小结

| 目标           | 策略                     | 工具例子                |
| -------------- | ------------------------ | ----------------------- |
| 启动 / 切换快  | 键盘优先、Raycast 脚本化 | Raycast, kitty          |
| 降低上下文切换 | 统一平台 / Mock / 翻译   | Apifox, Bob, Cursor     |
| 环境可复现     | Brewfile / Dotfiles      | Homebrew, mise          |
| 性能与轻量     | 选原生应用 / 低资源方案  | OrbStack, kitty, Zed    |
| 知识沉淀       | Markdown + AI 总结       | Obsidian, Cherry Studio |

## 快速新机初始化（思路）

1. 安装 Homebrew → 执行 `brew bundle --file=Brewfile` 恢复工具栈
2. 克隆 dotfiles 仓库（Zsh 配置 / alias / kitty 主题）
3. mise 安装运行时（Node / Python / Go）
4. Raycast 导入脚本 & 快捷键配置
5. 登录 1Password → 解锁 SSH Key / 密钥
6. OrbStack 启动开发用容器服务
7. 配置 Cursor / Zed 登录与插件同步

## 结语

保持工具栈精简：**能替换、可脚本、少依赖**。工具服务于效率，而非效率服务于工具。欢迎按需裁剪，或就某个工具进一步交流！
